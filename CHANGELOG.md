# CHANGELOG

All notable changes to EPSILON will be documented in this file.

---

## [v1.3]

### Added
- Delegation Management module
  - Update folder-level delegate permissions across common folders
  - Review folder-level delegate permissions
  - Review mailbox-level delegation (Full Access, Send As, Send on Behalf)
  - Downgrade mailbox-level delegates to folder-level Reviewer

### Enhanced
- Improved delegation visibility and control
- Better support for assistant / executive access workflows
- Simplified transition from mailbox-level access to scoped folder-level permissions

### Improved
- Continued improvements to script structure, reliability, and usability
- Maintained consistent menu-driven experience across modules

---

## [v1.2]

### Added
- Calendar Permissions module
  - Grant, update, and remove calendar access
  - View existing calendar permissions
  - Built-in access level selection
  - Automatic detection and updating of existing permissions

### Improved
- Delegation workflows for assistant / executive scenarios
- Handling of duplicate permission assignments
- Overall script reliability and usability

---

## [v1.1]

- Startup menu (Exchange vs Compliance)
- macOS PowerShell Core support
- Executable (.exe) version

### Improved
- Menu navigation and structure
- Inbox rule management
- Compliance purge workflow
